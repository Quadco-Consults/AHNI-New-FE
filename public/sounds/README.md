# Notification Sounds

## notification.mp3

Optional notification sound for chat messages. If this file is not present, the chat will work normally but without sound notifications.

You can add any short MP3 file here named `notification.mp3` for chat message notifications.

Recommended:
- Duration: 0.5-2 seconds
- Format: MP3
- Size: < 100KB

The sound notification feature gracefully handles missing audio files.
